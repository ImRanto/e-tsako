## À propos

**E-tsako** est un projet entrepreneurial malgache dédié à la production de produits alimentaires artisanaux, locaux et authentiques.  
Il valorise le savoir-faire traditionnel et les ingrédients locaux à travers une gamme de produits savoureux, fabriqués dans le respect des normes d’hygiène.

**Localisation** : Antananarivo – Madagascar

---

## Objectifs

### Court terme
- Augmenter la production
- Structurer la distribution locale

### Long terme
- Diversifier les produits
- Ouvrir un point de vente
- Lancer bien la boutique en ligne
